# Boogie Woogie Piano Analysis
Analysis of boogie woogie piano music extracted from youtube as wave files. The goal is to predict the number of net likes based on audio analysis.

## Python Environment
### Create

- `pyenv install 3.11`

- `pyenv virtualenv bwenv`

- `pip install -r requirements.txt`

### Activate

`pyenv activate bwenv`
